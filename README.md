# 🚦 Dynamic Traffic Control System using VLSI

## 📌 Overview
The Dynamic Traffic Control System using VLSI is an intelligent traffic management system that adjusts traffic signals based on real-time vehicle density. It helps in reducing congestion, waiting time, fuel consumption, and improves overall traffic efficiency.

## 🎯 Objectives
- Adjust traffic signals dynamically based on vehicle density  
- Reduce traffic congestion and delays  
- Provide priority for emergency vehicles  
- Detect traffic violations using sensors  
- Improve road safety and efficiency  

## 🧠 Working Principle
The system uses VLSI technology to process data from sensors such as IR sensors and sound sensors. Based on the inputs, a Finite State Machine (FSM) controls the traffic signals:
- High density → Longer green signal  
- Emergency vehicle → Immediate priority  
- Equal density → Predefined priority
