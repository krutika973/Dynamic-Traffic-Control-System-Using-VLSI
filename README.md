🚦 Dynamic Traffic Control System using VLSI
📌 Overview
The Dynamic Traffic Control System using VLSI is an intelligent traffic management solution designed to optimize traffic flow at intersections. It uses real-time data from sensors to dynamically adjust traffic signal timings, reducing congestion, waiting time, and fuel consumption.

🎯 Objectives
Adjust traffic signals based on real-time vehicle density
Reduce traffic congestion and delays
Provide priority for emergency vehicles
Detect traffic violations using sensors
Improve overall traffic efficiency and safety

🧠 Working Principle
The system uses VLSI technology to process inputs from:
IR sensors (traffic density detection)
Sound sensors (emergency vehicle detection)

Based on inputs, a Finite State Machine (FSM) controls traffic lights dynamically:
High density → longer green signal
Emergency detected → priority signal
Equal density → predefined priority
